# Get-MidiSessionList

## SYNOPSIS
This command enumerates all the active MIDI Sessions in the service

## SYNTAX

```
Get-MidiSessionList [<CommonParameters>]
```

## DESCRIPTION
This command enumerates all the active MIDI Sessions in the service.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-MidiSessionList
```

This command will output all current active MIDI sessions
