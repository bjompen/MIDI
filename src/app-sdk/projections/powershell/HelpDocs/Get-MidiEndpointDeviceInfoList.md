# Get-MidiEndpointDeviceInfoList

## SYNOPSIS
This command gets a list of all current MIDI endpoints.

## SYNTAX

```
Get-MidiEndpointDeviceInfoList [<CommonParameters>]
```

## DESCRIPTION
This command gets a list of all current MIDI endpoints.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-MidiEndpointDeviceInfoList
```

This command will return every current available MIDI endpoint
